Planning and Optimization Core

MDSLite's Planning and Optimization Core offers advanced planning, scheduling, and optimization using DELMIA Ortems.

Key Features:

  • Anticipate capacity needs by balancing production plans with load and projections.
  • Ensure continuous supply by synchronizing supplier production with customer demand.
  • Maximize production efficiency with optimized scheduling and management of unplanned events.

Execution Core

MDSLite's Execution Core streamlines manufacturing operations, providing real-time visibility and synchronization across production, quality, time, attendance, and warehouse management.

Key Features:

  • Order Dispatch: Ensures the right orders are processed by the right employees at the right time.
  • Order Execution: Real-time status updates reduce paperwork and improve organizational agility.
  • Electronic Work Instructions & Quality Data: Aligns with R&D requirements, instantly reflects engineering changes, and enables full traceability.
  • Downtime Classification: Tracks production losses and indirect labor, driving continuous improvement.

Manufacturing Intelligence and Analytics (MIA) Core

MDSLite's MIA Core provides real-time manufacturing intelligence, analytics, and dashboards, offering global visibility of OEE (Overall Equipment Effectiveness). Built on DELMIA Apriso – MPI and Dashboard Builder, it delivers key insights for efficient operations.

Key Features:

  • OEE Overview per Region: Standardized KPIs for global comparison across locations.
  • Factory Visualization: Real-time production events for greater transparency.
  • Manufacturing KPI Dashboards: Standardized dashboards that display performance losses and efficiency metrics in real time.

Machines and System Integration (MSI) Core

MDSLite’s MSI Core enables seamless integration with external systems and machines, built on DELMIA Apriso – Business Integrator.

Key Features:

  • ERP Integration: Connects with ERP systems like SAP ECC, S4 HANA, MS Dynamics, and legacy ERPs to exchange data.
  • PLM Integration: Syncs process plans, designs, and master data between PLM (Team Center, Enovia, SAP PLM, etc.) and MES.
  • APS Integration: Integrates master and transactional data between MOM and APS applications (ORTEMS, PREActor) to optimize order execution and scheduling.
  • Machines Integration: Real-time machine data collection using SCADA/OPC to automate OEE calculation and improve machine utilization.
